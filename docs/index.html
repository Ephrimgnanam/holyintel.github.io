About this project

This repository presents Henry A. Kissinger’s undergraduate thesis, “The Meaning of History: Reflections on Spengler, Toynbee, and Kant,” in multiple formats: a readable HTML edition, a generated mindmap for visual navigation, a derivative outline in Markdown, and a brief summary. The Markdown links open on GitHub’s site for proper rendering.

Provenance

The Mindmap and Summary were prepared by Jeffrey Emanuel using FixMyDocuments. You can also create similar mindmaps with the open‑source generator here: mindmap‑generator.
